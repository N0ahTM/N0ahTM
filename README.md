## Featured Project

### [AudioPlaybackConnector2](https://github.com/N0ahTM/AudioPlaybackConnector2)

A Windows system-tray app for connecting Bluetooth audio devices as A2DP audio sinks. It is built with **C++23**, **WinUI 3**, and **C++/WinRT**, packaged as an **MSIX**, and includes CI, CodeQL, releases, settings persistence, auto-reconnect behavior, and multi-language support.

This is the kind of project I like most: a focused tool that solves a real Windows workflow problem with native APIs and a clean user experience.
